# Hands-free-Douyin-scrolling  隔空刷抖音

## 项目描述

这是一个基于计算机视觉的自动化工具，通过手势识别技术实现手势隔空刷抖音。使用MediaPipe手部关键点检测，实时捕捉用户手势并模拟键盘操作，实现无需接触键盘即可刷抖音的功能。

## 功能特点

### ✨ 手势控制功能
- **上滑手势**：切换到下一个视频
- **下滑手势**：切换到上一个视频  
- **点赞手势**：为当前视频点赞

### 🎯 技术特性
- 实时手部关键点检测与可视化
- 智能手势识别算法
- 防误触机制和冷却时间控制
- 低延迟响应，流畅体验

### 启动程序

### 手势操作指南

#### 🤚 基本手势位置
- 将手置于摄像头视野范围内
- 保持适当距离（约30-50cm）
- 确保光线充足

#### 📱 具体手势说明

**1. 上滑手势（下一个视频）**
- 动作：食指向**上**快速滑动
- 触发：滑动距离 > 0.1，冷却时间1秒
- 效果：模拟按下键盘`↓`键

**2. 下滑手势（上一个视频）**
- 动作：食指向**下**快速滑动  
- 触发：滑动距离 > 0.1，冷却时间4秒
- 效果：模拟按下键盘`↑`键

**3. 点赞手势（点赞视频）**
- 手势：竖起大拇指 👍
- 触发条件：
  - 大拇指竖起（指尖在关节上方）
  - 手掌直立（手腕在小拇指上方）
  - 食指弯曲（指尖在关节下方）
- 冷却时间：5秒

## 使用技巧

### 💡 最佳实践
1. **手势清晰**：做手势时动作要明显、幅度适中
2. **保持稳定**：避免手部抖动，确保识别准确
3. **注意冷却**：每个手势都有冷却时间，避免频繁操作
4. **光线充足**：在光线好的环境下使用，识别更准确

### 🔧 故障排除
- **手势不识别**：检查摄像头是否被遮挡，调整手部位置
- **响应延迟**：关闭其他占用摄像头的程序
- **误触发**：调整手势识别距离阈值参数

## 注意事项

### ⚠️ 重要提醒
1. 使用前确保抖音已打开并处于活动窗口
2. 程序运行期间不要最小化摄像头窗口
3. 手势操作时保持手部在摄像头视野内
4. 冷却时间设置可防止误操作，请勿设置过短
